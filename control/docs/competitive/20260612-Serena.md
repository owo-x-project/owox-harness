# Serena

## 調査日

2026-06-12

## 分類

文脈管理。

MCP 経由で code の意味検索、編集、確認を行う道具群。

## 概要

Serena は、LLM を codebase agent にするための MCP 道具。

言語サーバーを使い、記号単位の検索、参照、編集、作業記憶を提供する。
owox-harness から見ると、target harness が使うべき文脈取得道具の有力候補。

## 公開値

2026-06-12 時点:

- GitHub 星数: 約 25.3k
- 分岐数: 約 1.7k
- 履歴数: 2,946
- MCP サーバーとして使える
- 言語サーバー連携あり

## 何がすごいか

- code を記号単位で扱う
- MCP で複数 AI CLI から使いやすい
- grep より意味に寄った文脈取得ができる
- 編集、検索、記憶を同じ道具群に置く
- 既存の言語サーバー能力を活かす

## owox-harness が取り入れる点

- target harness の文脈取得道具候補
- 記号単位の参照
- MCP 経由の道具公開
- 「全文投入しない」文脈設計
- 道具利用の記録

## owox-harness が超える点

### 目的の違い

Serena は文脈取得と編集の道具。
owox-harness は正本、task、検証、人間判断を管理する。

### 文脈の理由

Serena は必要な情報を取れる。
owox-harness はなぜその文脈が必要だったかを task と来歴へ結ぶ。

### target harness

Serena は単体道具。
owox-harness は target harness の中に道具利用方針を置く。

### 腐敗検知

Serena は検索と編集が強い。
owox-harness は文脈や task の古さを検出する。

## owox-harness が負けている点

- 意味検索
- 記号単位の編集
- MCP 道具体験
- 対応言語の広さ
- 実装済みの文脈圧縮

## 勝つための判断

Serena は敵というより採用候補。

owox-harness は Serena を置き換えない。
target harness の文脈取得道具として使うか、同じ考え方を取り入れる。

## 見直し条件

- Serena が作業流れや task 管理まで広げた時
- Serena が正本検証を持った時
- Serena が人間判断の来歴を持った時
- Serena が target harness 生成へ寄った時
