# Graphiti

## 調査日

2026-06-12

## 分類

記憶管理。

時間つき知識グラフで agent 記憶を扱う。

## 概要

Graphiti は、動的環境向けの時間つき知識グラフ。

情報の関係、時間、出典を扱い、agent の記憶や検索に使う。
owox-harness の経験保存、来歴検索、記憶分離で参考になる。

## 公開値

- GitHub 星数: 約 27.3k
- 分岐数: 約 2.4k
- 履歴数: 1,166

## 何がすごいか

- 記憶を時間と関係で扱う
- 出典を持つ記憶を意識している
- 動的に変わる情報を扱う設計
- MCP から使える
- agent 長期記憶の文脈で認知がある

## owox-harness が取り入れる点

- 記憶に時間を持たせる
- 出典つき検索
- 来歴検索との接続
- MCP 経由の記憶参照
- 変わった事実と古い事実を分ける考え方

## owox-harness が超える点

### 記憶の境界

Graphiti は記憶基盤。
owox-harness はドメイン固有情報と汎用経験を分ける。

### 判断の来歴

Graphiti は関係と時間が強い。
owox-harness は人間判断と完了理由を正本へ結ぶ。

### target harness

Graphiti は記憶を提供する。
owox-harness はどの記憶を target harness へ渡すかを決める。

### 検証

Graphiti は検索と記憶が強い。
owox-harness は記憶を根拠にした判断が検証済みかを見る。

## owox-harness が負けている点

- 長期記憶
- 時間つき知識グラフ
- 出典つき検索
- MCP 記憶連携
- 記憶基盤としての完成度

## 勝つための判断

Graphiti は「記憶を賢く検索する」で強い。
owox-harness は「どの記憶を使ってよいか、どの判断に使ったか」で勝つ。

記憶基盤は自作より連携候補として見る。

## 見直し条件

- Graphiti が task 管理へ寄った時
- Graphiti が人間ゲートや検証ゲートを持った時
- Graphiti が target harness 生成へ寄った時
- Graphiti が経験 export / import と重なった時
