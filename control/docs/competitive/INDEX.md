# competitive

競合、参考資料、超える対象を置く。

## 置くもの

- 競合調査方針
- 参照方針
- 調査候補
- 競合ごとの調査票
- owox-harness が取り入れる点
- owox-harness が超える点

## 置かないもの

- 採用判断そのもの
- 要件正本
- 実装計画
- 外部資料の長い写し

採用判断は `docs/decisions/` に置く。
要件は `docs/requirements/` に置く。

## 読む時

- 製品戦略を見直す時
- 差別化軸が弱い時
- 他 OSS の強みを取り入れる時
- 勝ち筋を確認する時
