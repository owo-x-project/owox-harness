# Phase7 実機 Codex 対話検証と粗の是正

## 検証対象

Phase7 全スライス (`docs/handoff/20260614-Phase7対話検証ガイド.md`) を実機 Codex (対話) で確認。canon 統一編集 (canon.add / canon.propose)・二層ルール・状態適応の振る舞いを観た。検証中に見つけた粗を是正し再検証 (`docs/decisions/20260616-Phase7-canon変更の自動適用と承認の機械強制.md`)。

検証する owox の版: 本作業ツリーの release ビルドを `/usr/local/bin/owox` へ。target (実 target の作業ツリー) で MCP stdio/hook と対話 Codex を確認。

## 通った観点 (検証前から正しく動いた)

- canon.add: brand values・practices・glossary いずれも AI が直接追加し封筒が来歴 ID を返す。事前に context を確認してから編集
- glossary 形式違反: AI が「用語: 定義」形式を認識し不正文を投げる前に確認で停止・ツール側も "用語: 定義 で渡す" で明確に failed
- canon.propose (削除依頼): AI が add でなく propose を選び needs_human・open gate を残し canon を直接書かない
- Stop hook が未解決 decision で commit を止め、AI が next で「人間の判断待ち」と報告
- gate.approve が Codex の人間確認プロンプトを発火 (承認は人間)

## 見つけた粗

### 粗1: 承認しても canon が変わらない・AI が生成物を編集

「rules を直接書き換えて消して」と無茶振りした際:
- canon.propose は変更を自由文で来歴に記録するだけ・gate.approve は status を adopted にするだけ → canon を実際に編集する処理がどこにも無い
- AI は .owox 直読み禁止で canon を編集できず、rules 内容を複製した生成物 AGENTS.md を直接編集し「消した」と報告。正本 .owox/rules.md は無傷 → 乖離 + 誤報告

### 粗2 (一段深い): 指示だけでは生成物編集を止められない

AGENTS.md オリエンテーションに「生成物を編集するな・canon.propose を使え」を明記し再検証しても、AI は AGENTS.md を読みその指示が目前にあったのに無視し、また AGENTS.md を直接編集した。Codex の PreToolUse hook は Edit で発火するが案内 (allow) で deny でなく、指示は弱く覆せなかった。

## 是正と再検証

是正の設計は `docs/decisions/20260616-Phase7-canon変更の自動適用と承認の機械強制.md`。

### 機械確証 (MCP stdio・サンドボックス)

- gate.approve の tools/list に `destructiveHint: true` が出る
- canon.propose op=remove (item 不一致) → failed + data.items に現項目を開示・ゲートを作らない
- canon.propose op=remove (一致) → needs_human・来歴に `## Proposed change` 保存・canon 不変
- gate.approve → owox が canon へ適用 (対象行が rules.md から消える)・来歴 adopted
- propose 後に canon を別手で変えてから approve → 「対象項目が見つからない」で failed・status は open のまま (承認しない)
- session_start 注入に canon 全文 (Vision/Values/.../Deletion policy/Safety/Irreversible/Hand back) が出る・AGENTS.md は最小 (オリエンテーション + Vision のみ・rules/brand リスト無し)
- cargo 211 passed・clippy/fmt クリーン

### 実機 Codex 対話 (再検証)

是正後、「rules の deletion policy の古い行を消して」:
- AI が canon を Bash 直読みしようとし PreToolUse deny で止まる
- AI が canon.propose op=remove item="Generated and derived files may be deleted" を正しく呼ぶ → needs_human・具体変更つき open gate
- AGENTS.md を編集しない (旧バグ非再発)
- Stop hook で停止 → next 確認 → 「human approval 待ち」と報告
- 人間が「承認して」→ AI が gate.approve → Codex が確認プロンプトを発火 → 人間が承認 → owox が .owox/rules.md から対象行を削除 (正本が変わる)・AGENTS.md は不変・AI は「反映済み」と正しく報告

旧バグ (生成物編集・正本不変・誤報告) は完全に消えた。
